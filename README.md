# Personal-Portfolio
A personal portfolio as a small web development project using HTML, CSS, and JavaScript.

## Features
- Dark theme with purple/violet/indigo glassmorphism accents
- Smooth scroll-reveal animations on each section
- Sticky navbar with active-link highlighting
- Skills section with real tech logos
- Flip-card project gallery with GitHub links
- Downloadable resume (PDF)
- Fully responsive
- Links leading to platforms like GitHub, LeetCode and HackerRank

## Tech Stack
- HTML5
- CSS3 (custom properties, backdrop-filter, grid/flexbox)
- JavaScript (IntersectionObserver for scroll effects)
- Devicon for skill icons
